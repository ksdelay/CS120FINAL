Instructions To run:
Chose 1 for text based project, 2 for graphics project
	Text based displays all test cases and gives you option to load game

	Graphics based prompts user if they would like to load a previous game, choose Y for yes N for no
	If you make a new game, enter your user name then the start menu should display
	Chose the menu option you want
	Once game is started press escape to quit
	"P" to pause
	Move using the left and right arrow keys
	Press spacebar to shoot
	Move your character left and right and shoot the falling asteroids
	Smaller yellow squares add points to your score and the 2x squares multiply your score
ENJOY!
